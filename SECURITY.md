# Política de Seguridad (Security Policy)

## Versiones Soportadas

Actualmente, solo la última versión mayor del Boilerplate recibe actualizaciones de seguridad.

| Versión | Soportada          |
| ------- | ------------------ |
| v1.12.x | ✅ Sí              |
| < v1.12 | ❌ No (Deprecada)  |

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, por favor **NO** crees una *Issue* pública. 
Utiliza el sistema de [Private Vulnerability Reporting](https://github.com/merci-boilerplateDF/PROYECTO_merci-boilerplate.es/security/advisories/new) de GitHub para reportarlo de forma confidencial a la autora. Te responderemos a la mayor brevedad posible para coordinar un parche antes de su divulgación.
